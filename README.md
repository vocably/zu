# Zulu

This repository contains the Zulu words and translations used by [Vocably](https://getvocably.com) - an open-source dictionary and language-learning tool available as:

- Browser extensions for [Desktop Chrome](https://chromewebstore.google.com/detail/vocably/baocigmmhhdemijfjnjdidbkfgpgogmb), [Desktop Safari](https://apps.apple.com/app/vocably-for-safari/id6464076425), and [iOS Safari](https://apps.apple.com/app/vocably-pro-language-cards/id1641258757)
- Mobile application for [iOS](https://apps.apple.com/app/vocably-pro-language-cards/id1641258757) and [Android](https://play.google.com/store/apps/details?id=com.vocablypro)

The repo files are automatically updated every five minutes.

Most of the files in this repo are generated by AI. AI isn't perfect, so some words or translations may not be correct.

You are more than welcome to contribute to this project by fixing poorly generated words and translations.

Lacking time or energy to create PRs? Join the [Discord server](https://discord.gg/Wd9hujvD) and report the problem.
